Pengujian Cache Proxy

Halaman ini dikhususkan untuk menguji sistem caching.

  • Request 1: Proxy akan mencetak log CACHE MISS, mengambil file ini dari Web Server, lalu menyimpannya di folder cache lokal.
  • Request 2: Proxy akan mencetak log CACHE HIT dan langsung mengirim salinan file ini ke Client tanpa menghubungi Web Server lagi.
